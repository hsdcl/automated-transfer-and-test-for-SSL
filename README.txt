The codes are run on Python with a version of 3.9

Instructions for use
-------------------------------------------------------
Codes in "AutoTransfer" folder are software for automated transfer structural superlubricity flakes to target positions.
Codes in "AutoTest" folder are software for measure the tribological performance of graphite flakes.

The software can be triggered by running the main.py script in each file.

Other ".py" files are components used to control.


Demo
-------------------------------------------------------
The demo dataset is presented in './Demo/data'
The expected output is presented in './Demo/Output'
Enter the "AutoTransfer" Folder
Run demo by "python ./vision/yolov5/detection.py"
The expected run time for demo is within 1 min.



Installation Guide
-------------------------------------------------------
Download the codes and create python environment with a version of 3.9
Install related python packages in Software Dependencies
Enter the "AutoTest" or "AutoTransfer" folder
Run “python main.py” to start the software
The typical install time for installation is about 10 mins.
